# Booksmart
Bookmark manager app - Save your favorite web sites.

![](https://i.ibb.co/DfcKT1Tp/booksmart.png)

## Features
- User authentication 
- Create collections and Tags and add them to your bookmark
- Search bookmark by url, title
- Modern design interface 
- Responsive and adaptable even on mobile

## Technologies
- Vite - Vue js as frontend framework
- Supabase as backend service
